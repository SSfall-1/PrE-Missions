### 🛠 Установка
1. Скачайте необходимый `.pbo` файл.
2. Поместите его в корневую папку игры по пути:
   `\Documents\Arma 3 - Other Profiles\ваш профиль\mpmissions`
3. Запустите сервер и выберите миссию в списке.

### ⚠️ Требования
Все миссии созданы на пресете эклипса

### 💻 Разработка и Contribution
Если вы хотите внести правки в скрипты или логику:
1. Клонируйте репозиторий.
2. Для исправлений используйте систему **Pull Requests**.
3. При написании скриптов (`.sqf`) придерживайтесь стандартов оптимизации для сохранения высокого FPS на сервере.
