### 🛠 Установка
1. Скачайте необходимый `.pbo` файл.
2. Поместите его в папку игры по пути:
   `\Documents\Arma 3 - Other Profiles\ваш профиль\mpmissions`
3. Запустите сервер и выберите миссию в списке.
